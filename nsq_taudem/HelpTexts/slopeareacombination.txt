#Help text for tool "Slope Area Combination".
URL
https://hydrology.usu.edu/taudem/taudem5/help53/slopeareacombination.html

TEXT
<p>Creates a grid of slope-area values = (S^m)(A^n) based on slope and specific catchment area grid inputs, and parameters m and n. This tool is intended for use as part of the slope-area stream raster delineation method.</p>
END TEXT
